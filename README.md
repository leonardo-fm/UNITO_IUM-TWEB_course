# IUM-TWEB
IUM-TWEB 2023-2024 Assignment UniTo
Project for the UNITO corse <i>[IUM](https://informatica.i-learn.unito.it/course/view.php?id=2838) and [TWEB](https://informatica.i-learn.unito.it/course/view.php?id=2734)</i> 2023/2024

Initialize angular frontend
go to solution/folder frontend;
exec the command: npm install
if not installed, install for build: npm install -g @angular/cli

Initialize main and express server
go to solution/backend/express
exec the command: npm install
go to solution/backend/main
exec the command: npm install
